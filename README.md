# Tanisha Sinha — Portfolio

AI/ML Engineer portfolio built with React + TypeScript + Vite.

**Live site:** https://tanisshaaa.github.io/Portfolio/

## About
Personal portfolio showcasing ML/AI projects, work experience at 4th Orbit, and research published at TENSYMP 2024.

## Tech Stack
React 19 · TypeScript · Vite · Framer Motion · Three.js · Tailwind CSS · Lenis

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
